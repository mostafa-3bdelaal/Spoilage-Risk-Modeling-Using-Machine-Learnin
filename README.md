# 📦 Cold Chain Spoilage Risk Prediction

## 📌 Project Description
This project predicts spoilage risk in cold chain logistics using machine learning. It analyzes environmental, transportation, and packaging factors to estimate the Spoilage_Risk_Score and identify key factors affecting product quality.

## 📊 Dataset
The dataset includes:
- Temperature and humidity conditions  
- Transit duration and vehicle load  
- Packaging quality and handling factors  

## ⚙️ Workflow
- Data loading and cleaning  
- Exploratory data analysis (EDA)  
- Feature engineering  
- Scaling and dimensionality reduction (PCA)  
- Training regression models  
- Model evaluation and comparison  

## 🤖 Models Used
- Linear Regression  
- Decision Tree Regressor  
- Support Vector Regressor (SVR)  
- Random Forest Regressor  

## 📈 Results
Models were evaluated using MAE, RMSE, and R² score. Random Forest performed best due to its ability to capture non-linear relationships.

## 🧠 Conclusion
Machine learning helps predict spoilage risk effectively and improves decision-making in cold chain logistics by identifying critical influencing factors.
